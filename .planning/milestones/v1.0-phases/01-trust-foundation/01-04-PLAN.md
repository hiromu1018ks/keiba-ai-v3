      - 入力 `races` を `race_start_datetime` 昇順でグループ化し、ユニーク race_id を時系列順に並べる
      - `for k in range(1, n_splits + 1)` で `test_start = int((k / (n_splits + 1)) * n)`, `test_end = int(((k + 1) / (n_splits + 1)) * n)`, `train_rids = unique_races[:test_start]`, `test_rids = unique_races[test_start:test_end]` を計算
      - 各 fold で以下の3つのガードを順に `raise ValueError` で実行（`assert` ではない）:
        1. `if not set(train_rids).isdisjoint(set(test_rids)): raise ValueError(f"race_id leakage across train/test in fold {k}")`
        2. `train_time_max = races.loc[races.race_id.isin(train_rids), "race_start_datetime"].max()`, `test_time_min = races.loc[races.race_id.isin(test_rids), "race_start_datetime"].min()`。`if not (train_time_max < test_time_min): raise ValueError(f"chronological boundary violated in fold {k}: train_max={train_time_max} >= test_min={test_time_min}")`（**strict `<`、等値は不可**）
        3. `if len(train_rids) == 0 or len(test_rids) == 0: raise ValueError(f"empty train or test in fold {k}")`

      - **同一 `race_start_datetime` を持つ複数レースの取り扱い（HIGH #2 document）:** 呼出元は等値タイムスタンプのレース群が train と test に跨らないよう、事前に `race_id` をタイスタンプで安定ソート（tie-break を `race_id` 辞書順で）してから本関数に渡す前提。等値タイムスタンプが fold 境界に掛かった場合、上記ガード2 がそれを検出して `raise ValueError` する。つまり本関数は**等値タイムスタンプの跨ぎを許さない**（strict `<`）。
